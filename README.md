# mc-archieve-server
Minecraft Archive + Achieve

Minecraft Archieve app backend repository

## Commands
- Setup docker db
```shell
docker compose -f docker-compose.dev.yml up
```

## Tech Stack
- Java 17
- Spring
  - Spring Boots
  - Spring Security
  - JPA
- JUnits
- MySQL
- S3 & R2
- Docker

## How to setup development environment
Set the active profile to `dev`.
